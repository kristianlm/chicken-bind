
* chicken-bind

A utility to create bindings to C/C++ in Chicken Scheme. This is an
unofficial clone of [[http://wiki.call-cc.org/eggref/4/bind][this]] Chicken Egg.  Sometimes, patches from this
repo end in upstream.
